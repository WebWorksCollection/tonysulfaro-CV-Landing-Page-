Computer Science Coursework

CSE 231 - Python

This was my first programming class at Michigan State in Python. I learned quite a bit about data structures and ingesting data from a flat file. The last two projects were games which were pretty cool because they utilized classes to store game data.

CSE 232 - C++

Overall, this is similar in structure to CSE 231 but in C++! One of the most challenging things I learned was how to use pointers and preventing memory leaks in my programs. Still the same flat file IO but using vectors and then with my own vector class and linked lists.

CSE 260 - Discrete Structures

This class gave me even more experience with computational theory including topics like data structures, algorithms, and finite state automata.

CSE 331 - Algorithms and Data Structures

Who doesn't love some algorithms? This Python course focused solely on analyzing, designing, and implementing algorithms and making them more efficient. Some extra math is mixed in here as well.

CSE 335 - Object-Oriented Software Development

Utilizing C++ this class focuses solely on the design and implementation of large programs using OOP.
